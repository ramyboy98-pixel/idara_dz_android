# مرحلة صورة النموذج

تم اعتماد نظام جديد بدل DOCX/Word:

1. يرفع المستخدم صورة واضحة للنموذج بصيغة PNG أو JPG أو WEBP.
2. يضيف حقول الاستمارة مثل: الاسم واللقب، العنوان، رقم الهاتف، التاريخ.
3. يفتح التطبيق صفحة تحديد الأماكن.
4. يختار المستخدم الحقل ثم يضغط على مكانه فوق صورة النموذج.
5. عند إنشاء PDF، يستعمل التطبيق الصورة كخلفية A4 ويضع القيم فوقها حسب الإحداثيات المحفوظة.

ملاحظة: دعم PDF كخلفية مباشرة يحتاج مرحلة لاحقة لتحويل صفحة PDF الأولى إلى صورة داخل التطبيق. هذه النسخة تبدأ بالصور لأنها مستقرة وقابلة للاختبار فورًا.
