# دليل ربط استمارة المعلومات بالنموذج

حتى يعرف التطبيق أين يضع المعلومات داخل النموذج، يجب وضع رموز داخل ملف النموذج نفسه.

مثال:

```text
أنا الممضي أسفله {{الاسم_واللقب}} الساكن بـ {{العنوان}} أتشرف بتقديم طلبي إلى {{الجهة_المستقبلة}}.
```

عند إنشاء استمارة المعلومات داخل التطبيق، أضف الحقول بنفس المعنى:

- الاسم واللقب  → الرمز الذي يظهر داخل التطبيق: `{{الاسم_واللقب}}`
- العنوان → `{{العنوان}}`
- الجهة المستقبلة → `{{الجهة_المستقبلة}}`
- تاريخ الطلب → `{{تاريخ_الطلب}}`

عند ملء الاستمارة لاحقًا، سيستبدل التطبيق هذه الرموز بالقيم الحقيقية ثم يصدر PDF.

ملاحظة: في المرحلة الحالية أضفنا طريقة إظهار الرموز وحفظ الحقول. في المرحلة التالية سنضيف صفحة ملء الاستمارة، وبعدها محرك استبدال الرموز وتصدير PDF.
